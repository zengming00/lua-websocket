# lua-websocket
websocket written in pure Lua, Can be run on openwrt

```shell
opkg update
opkg install lua-copas lua-coxpcall luabitop luasocket lua

lua websocket-server.lua
```

test in OpenWrt 21.02.0-rc2 r16122-c2139eef27
